# 上手Ravenfield

## 启动选项

我们默认您有一点点的steam使用经验

点击steam库的`开始游戏`，会出现一下选项：

| 选项名 | 操作 |
|------|------|
| 启动RF | 正常启动游戏 |
| 启动RF (No mods) | 启动游戏而不自动加载除地图外的mod |
| 启动RF能效测试工具 | 运行游戏效果跑分测试（没什么卵用说是，毕竟原版体验不代表加mod之后的表现） |
| 启动RF (Safe mode) | 启动游戏并重置游戏窗口大小 |

按需选择即可，可在steam的游戏属性处修改默认选项

### Mod管理

点击`MODS`，打开mod管理界面，你可以在这里看到你“订阅”的mod

点击右上角的`Browser workshop`会跳转到steam创意工坊

点击项目图标可以禁用mod（需要重启游戏或点击右下角的`RELOAD MODS`重新加载mod生效），点击一个项目的`🔧`图标会跳转到这个项目的工坊页面

::: tip 关于“工坊”
在steam中类似于模组中心的功能叫做“创意工坊”

以工坊项目为一个单位（可以理解为一个mod包），一个工坊项目里可以包含一个或多个类型的单独mod

“订阅”一词在这里可以简单理解为为向steam提出下载、保存到玩家配置并自动更新一个工坊项目的请求
:::

### 设置

### 游戏模式
接上一章

游戏模式分为以下三个：

| 模式 | 描述 |
|------|------|
| Instant Actions 即时行动 | 一般模式 |
| Conquest 征服 | 战旗模式，已经暂时退出游戏 |
| Arcade 街机 | 卡牌策略与一般模式的结合 |

在实际中，我们主要游玩的是INSTANT ACTION模式，以下的文档内容主要面向此模式

选择INSTANT ACTION模式，我们将配置这局游戏

### 对局配置

点击INSTANT ACTION进入本局游戏的配置界面

在左上角的大封面中选择地图，可搜索或筛选、调整预览图标大小

在`GAME MODE`处选择游戏模式，目前游戏有以下6个模式：
- `SKIRMISH` 死斗模式
  双方只有一个初始据点，玩家无法重生（飞机增援时除外），死亡后将附身在存活的AI，继承AI的武器或者载具继续战斗。双方队伍拥有2次增援，双方兵力将不断损耗，直至一方兵力损耗完或者把地图旗帜据点全部占领，游戏结束。
- `POINT MATCH` 征服模式
  双方任意一方达到击杀数量即胜利
- `BATTLE` 对战模式 
  游戏开始时，双方都有固定兵力，击杀敌人或占领了地图内全部据点都会损耗敌方兵力，如果敌方兵力降为0后游戏胜即利。
- `SPEC OPS` 特别行动模式
  特工任务模式
  
  懒(✿◡‿◡)，此处直接引用，参见[Steam指南](https://steamcommunity.com/sharedfiles/filedetails/?id=2584237725)（感谢 @[VadertheConqueror](https://steamcommunity.com/profiles/76561199102628353)）
- `HAUNTED` 亡灵模式
  类似于生存模式，你需要带领队员在骷髅的围剿攻击下生存7回合即可胜利。骷髅分为2种，一种是近战骷髅，一种是远程骷髅，每回合都会有新的武器补给，但是攻击的时候不能跑出圈外，不然会持续掉血。
- `DOMINATION` 兵力对抗模式
  游戏双方都有三个进程，一方三个进程全部结束，则该方不会再继续刷兵，并且游戏会胜利，如果一方据点占领得多，血条掉的也会更快。

`PLAYER TEAM`处选择玩家的队伍，`Eagle`和`Raven`分别为蓝、红队，`Spectator`为旁观模式（即上帝视角，游戏中你可以通过单击一个AI来追踪他的行动，再次单击取消）

`GAME LENGTH`处选择对局长度，`SHORT`为短，`EPIC为超长`，以此类推（说白了就是这局游戏你想玩多久，在一些游戏模式中该选项也等于游戏难度）

`BOT AMOUNT:`处选择一局游戏中，双方各自最多同时存在几只AI

`Night`可以让地图时间变为夜晚（即天黑），
`Respawn Time`可以调整重生的缓冲时间，
`Configure Flags`可以在游戏开始时调整地图中双方的初始旗帜，

接着翻到下一页（CONFIG页），你可以调整更多

这一页可以预览队伍配置

在`SAVE`处上面的按钮选择已保存的游戏配置（即脚本、武器、皮肤、载具等配置）（如果你下载了一些整合包也是在这里选择），也可以安`SAVE`来保存自己的配置，方便调用。（重命名时如果名字已存在会覆盖已有的配置）

点击`EDIT EAGLE`或者下面红队的选项，来修改队伍配置

最顶处选择本队伍的预设配置

`TEAM NAME`和`COLOR`修改队伍名字和颜色

`CHARCTER SKINS`处选择双方的皮肤（上方的窗口可以预览，载具、武器同样）。

单击`WEAPON SELECT`可以选择AI可用的武器，右上角可以搜索、选择分类和筛选，每个图标左上角的小圆点表示稀有度（玩家和bot可以获得使用的概率，包括禁用、普通、稀有和超稀有）

`PAINT ALL`为应用左上角四个圆点中被选中的稀有度到所有武器

单击`VEHICLE SELECT`可以选择AI队伍的载具

单击`MUTATORS`可以选择启用的脚本，或设置脚本。脚本可以更改游戏的HUD皮肤，或禁用掉友方伤害（没错这游戏有友方伤害）等

游戏自带一些脚本包括玩家无武器清单配置、巧克力雨菜单、击杀提示、血液溅射爆炸、限制地对空打击等，其他的可以从创意工坊中获取。


## 游戏操作
### 基本操作

当你配置好对局后，点击右下角的`PLAY`，游戏加载完地图后，会进入武器选择界面。

左边的武器框单击可选择武器

武器选择界面中的`NOTHING`的意思是该槽位不选择武器。

武器选择界面上方的是标签用来分类武器，某些MOD作者会专门给自己的武器做一个标签

游戏默认TAG标签：
| TAG | 译名 |
|------|------|
| CLOSE QUARTERS | 近战武器 |
| ASSAULT | 突击性武器 |
| MARKSMAN | 远程武器 |
| EQUIPMENT | 装备 |
| GRENADES | 手榴弹 |
| ANTI ARMOR | 反装甲武器 |

而左边的`DEPLOYMENT`可以选择出生、重生点，`BATTLE PLAN`的作用在[指挥系统](#指挥系统)中会讲。

选择完后，点击`DEPLOY`键，稍作等待即可进入游戏。

当你进入战斗后，你就可以尽情的与队友干翻红队了(＾０＾)

这是游戏的默认键位：

| 功能 | 对应按键 |
|------|------|
| 开火 | 鼠标左键|
|瞄准 | 鼠标右键|
|武器切换 | 鼠标滚轮或1~5数字键|
|移动 | WASD|
|侧斜头 | Q或E|
|子弹时间（慢动作） | Caps Lock|
|打开玩家列表/计分板 | Tap|
|夜视仪 | N|
|跳跃或打开降落伞 | 空格|
|打开地图 | M|
|打开\关闭UI | Home和End键|
|发射载具干扰弹 | X|
|踢（踢尸体掉弹药包） | V|
|交互（爬楼梯、使用载具......） | F|
|载具第三视角 | C|
|选择载具座位 | F1~F7键|
|暂停 | Esc|

这是游戏中的UI：

当然还有玩家列表（TAB键）处“`K`、`D`、`F`、`V`”，分别表示击杀数（误杀友军会抵消）、死亡次数、占领据点数、击毁载具数

游戏中按Enter键选择武器后可以点击`RESPAWN`重生并应用。

`Esc`键可以暂停游戏并打开菜单。

`RESUME`是继续游戏

`RESTAET`是重新开始游戏

`OPTIONS`是游戏选项（跟游戏主界面的选项一样）

`QUIT TO MENU`是回到主界面

`SHUTDOWN`是关闭游戏


游戏中按`F8`键可以暂停游戏并切换上帝视角，操作与一般游玩时一样，操作增加了按`Q`、`E`升降高度、`滚轮`调整移动速度，再按一次`F8`退出视角
### 指挥系统

终于讲到指挥系统了

\^o^/
指挥系统分为全局指挥和小队指挥。

#### 全局（团队）指挥
全局（团队）指挥就是我们刚才在武器选择界面的的`BATTLE PLAN`，点击进去，会看到这样一个界面：


要指定进攻方向，就要将鼠标移至一个据点，长按左键拉到要进攻的据点，右键标记取消标记。

要指定那些据点需要防御，就右键一个据点，右键取消标记。战斗中也按`Enter`键可以进行同样的操作，操作玩后按`Enter`键退出，注意不要按`RESPAWN`。

将左上角的`ALWAYS FOLLOW PLAN`点上可以让AI总是执行这个计划（否则这个部署的计划会一直执行到执行这个计划的AI死亡）。

此指挥方式对全部友军有效


#### 小队指挥

在指挥自己的队伍之前，你必须有一个属于自己的小队吧？（⊙ｏ⊙）

要招募队员，对着AI点击G键就可以了。

在讲解如何指挥前，我们先讲讲HUD处的队友状态和数量

回到游戏中的HUD处（左、右下角）

`SQUAD(2)`的意思是小队中有2个队友

`NO SQUAD`的意思是你的小队内没有队员

横杠后面的是队友的状态

`FOLLOWING`是跟随中，此时队友会一直跟着你

`MOVING`是移动中，代表队友正在去往指定地点的路上

`HOLDING`是待命中，即队友在指定地点待命中或防御指定据点

下面我们就来讲讲指挥的两种方式：

通过`G`键

首先，长按`G`键，屏幕中央出现一个框

框中`GO TO`指向你的队友去的指定位置

长按`G`的同时你也会发现屏幕中出现了一些点，点中带旗帜的是据点，没有的是载具

你可以将准星对准据点下达命令让队友去占点（前提是非己方据点）

准星对准据点后松手，队友会前往此点并占领

要让队友防守某点，操作与上面一样，长按`G`并将准星对准要防守的据点

在队友待命时，长按`G`并将视角移至脚底可以召集队友，单击`G`键也一样

还有针对一个敌方单位的操作

长按`鼠标右键`同时长按`G`键，可以指挥队员针对性攻击一个敌方单位


要让队友上载具，就这样上面一样，按长`G`将准星对准载具：

要让队友下载具，同上：

那`Commander ***`是什么意思？

命令的意思是招募载具内友军和即将要上载具的友军加入你的小队

通过面板指挥

点击`B`键，会打开一个面板，内有一个地图

左下角的`REGROUP`也是召集队友

直接在地图任何一处地方点一下左键，这时地图上会出现一个箭头，你的队友就会前往此地

你也可以在地图上右键任何一个据点，会出现几个选项

`Move to` 单纯让队友过去

`attack point` 过去并占领此点

如果是己方的据点，`attack point`会变成`defend point`（防守此点）

### 战役模式

::: tip
该模式已退出新版本
:::

因为战役模式目前来说尚不成熟，所以本篇仅对战役模式的玩法进行简单介绍

讲到战役模式

玩家需要在每回合游戏中尽可能的获取游戏的胜利

并不断升级自己的科技树以对付不断强大的对手

每个据点被占领后每回合都可以产出金钱或科技点数（升级科技树可以用到）

下面就是操作介绍

这是存档加载界面

`NEW`的意思是新建存档，`CONTINUE`的意思是继续上一次的存档

进入游戏后，会进入这个界面

中间的是战局指挥台，右边的是点科技树的地方

在游戏的某些界面时，单击右键可以返回到上一级界面

战局指挥台的操作

在某一据点单击，点击`Recruit`可以生成兵力（只有某些点可以，生成兵力需要金钱）

点击某一据点，据点的蓝色小人向上弹起（多按1次会多弹起一个），这时选中一个据点，点击`CAPTURE`或`MOVE`（`MOVE`的出现前提是该据点是本方据点），可以移动这些兵力（每回合一个兵力只能移动一次），未被占领的据点会被占领，如果要攻击一个据点的敌人，操作一样，但是选中兵力后是点击`ATTACK`

当你被敌人攻击时或攻击敌人时会出现这个界面

`AUTO RESOLVE`可以让AI直接自动战斗

`START`的功能类似实时对局模式，点击后玩家会进入战斗

`CANCEL`的功能是你攻击敌人时的撤销动作，防止不必要的手误（被敌人攻击时该选项不可用）

升级技能树

在全局视角时，鼠标点击右边的工作台以升级科技树

有两个科技树，分别是武器和载具

点击一个项目以进行操作

鼠标拖动可以移动视角

点击一个选项会弹出一个框是关于武器的介绍

下边的`UNLOCK`点击可以解锁该选项，`FOR`后面跟着的是解锁该选项的条件
