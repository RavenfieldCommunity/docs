# 上手Ravenfield

## 启动选项

我们默认您有一点点的steam使用经验

点击steam库的`开始游戏`，会出现一下选项：

| 选项名 | 操作 |
|------|------|
| 启动RF | 正常启动游戏 |
| 启动RF (No mods) | 启动游戏而不自动加载除地图外的mod |
| 启动RF能效测试工具 | 运行游戏效果跑分测试 |
| 启动RF (Safe mode) | 启动游戏并重置游戏窗口大小 |

按需选择即可，可在steam的游戏属性处修改默认选项

### Mod管理

点击`MODS`，打开mod管理界面，你可以在这里看到你“订阅”的mod

点击右上角的`Browser workshop`会跳转到steam创意工坊

点击项目图标可以禁用mod（需要重启游戏或点击右下角的`RELOAD MODS`重新加载mod生效），点击一个项目的`🔧`图标会跳转到这个项目的工坊页面

> 在steam中类似于模组中心的功能叫做“创意工坊”，工坊项目为一个单位（可以理解为一个mod包），一个工坊项目里可以包含一个或多个类型的mod
>
> “订阅”一此在这里可以简单意为向steam提出下载与自动更新一个工坊项目

#### 对局配置

点击INSTANT ACTION进入本局游戏的配置界面
在MAPS处的下拉菜单中选择游戏的自带地图，CUSTOM MAP处选择你从创意工坊中下载下来的地图或地图编辑器中保存的地图。

在GAME MODE处选择游戏模式，目前游戏有以下6个模式：
SKIRMISH【死斗模式】
该模式下双方只有一个初始据点，玩家无法重生（飞机增援时除外），死亡后将附身在存活的AI，继承AI的武器或者载具继续战斗。双方队伍拥有2次增援，双方兵力将不断损耗，直至一方兵力损耗完或者把地图旗帜据点全部占领，游戏结束。
POINT MATCH【征服模式】
此模式双方任意一方达到击杀数量即可胜利。
BATTLE【对战模式】
该模式特点为，游戏开始时，双方都有固定兵力，击杀敌人或占领了地图内全部据点都会损耗敌方兵力，如果敌方兵力降为0后游戏胜即利。
SPEC OPS【特别行动模式】
懒(✿◡‿◡)，此处直接引用（感谢 @VadertheConqueror）

SPEC OPS模式超详细指南（EA26大更新！） 
A Guide for Ravenfield
By: mambaout
spec ops 模式，即特别行动模式，是RAVENFIELD众多比赛模式中较有特色的一种 由于没有任何教程，许多玩家对此模式可能感到困惑 本指南旨在提供特别行动模式的介绍、打法及武器和mod的选择，应该是全网最全了 有用请点个赞谢谢

HAUNTED【亡灵模式】
类似于生存模式，你需要带领3名队员在骷髅的围剿攻击下生存7回合即可胜利。骷髅分为2种，一种是近战骷髅，一种是远程骷髅，每回合都会有新的武器补给，但是攻击的时候不能跑出圈外，不然会持续掉血。
DOMINATION【兵力对抗模式】
游戏开始时，游戏双方都有三个进程，一方三个进程全部结束，则该方不会再继续刷兵，并且游戏会胜利，如果一方据点占领得多，血条掉的也会更快。


PLAYER TEAM处选择玩家的队伍，Eagle和Raven分别为蓝、红队，Spectator为旁观模式（即上帝视角，游戏中你可以通过单击一个AI来追踪他的行动，再次单击取消）

GAME LENGTH处选择对局长度，SHORT为短，EPIC为超长，以此类推（说白了就是这局游戏你想玩多久，在一些游戏模式中该选项也等于游戏难度）

BOTS处选择一局游戏最多同时存在几个AI

BALANCE处调整游戏双方人数平衡，具体的AI数取决于BOTS的值，圆圈靠近哪方哪方人数相对更多


接着往下拉，你可以调整更多
在LOAD CONFIGURATION下拉菜单处选择已保存的游戏配置（即脚本、武器、皮肤、载具等配置）（如果你下载了一些整合包也是在这里选择），也可以通过SAVE NEW CONFIG来保存自己的配置，方便调用。（重命名时如果名字已存在会覆盖已有的配置），删除配置请参考4.1游戏Q&A


CHARCTER SKINS处选择双方的皮肤（上方的窗口可以预览，载具、武器同样）。

单击WEAPON SWITCH可以选择AI可用的武器，右上角的TOGGLE ALL的功能是全部启用\全部不启用，BOTHTEAMS处的下拉菜单可以分别为Eagle和Raven两队分别启用武器（按颜色判断），默认为为所有的AI启用\关闭。

单击VEHICLE SWITCH可以选择AI队伍的载具，两个大框分别对应两个队伍，点击一个小框可以选择载具，最上方的Load Preset可以一键启用MOD包的载具，Official Vehicle为游戏自带载具，None为无载具（即禁用载具）。

单击MUTATORS SWITCH可以选择启用的脚本，或设置脚本。脚本可以更改游戏的HUD皮肤，或禁用掉友方伤害（没错这游戏有友方伤害）等，游戏自带一些脚本，其他的可以从创意工坊中获取。

## 游戏操作
### 基本操作

当你配置好对局后，点击右下角的PLAY，游戏加载完地图后，会进入武器选择界面。
左边的武器框单击可选择武器
武器选择界面中的NOTHING的意思是该槽位不选择武器。








武器选择界面上方的是标签用来分类武器，某些MOD作者会专门给自己的武器做一个标签
ASSAULT意为突击性武器
MARKSMAN是远程武器
CLOSE QUARTERS是近战武器
STEALTH是消声型的武器（≈特别行动模式专用？）
GRENADES是手榴弹
ANTI ARMOR是反装甲武器（打坦克一类的）
EQUIPMENT是装备（如弹药包一类）


而左边的DEPLOYMENT可以选择出生、重生点，BATTLE PLAN的作用在2.2指挥系统中会讲。

选择完后，点击DEPLOY键，稍作等待即可进入游戏。

当你进入战斗后，你就可以尽情的与队友干翻红队了(＾０＾)

全局指挥
全局指挥就是我们刚才在武器选择界面的的BATTLE PLAN，我们点击进去，会看到这样一个界面：





















那么如何指挥呢？¯\(°_o)/¯
要指定进攻方向，就要将鼠标移至一个据点，长按左键拉到要进攻的据点，右键标记取消标记。





















要指定那些据点需要防御，就右键一个据点，右键取消标记。战斗中也按Enter键可以进行同样的操作，操作玩后按Enter键退出，注意不要按RESPAWN。
将左上角的ALWAYS FOLLOW PLAN点上可以让AI总是执行这个计划（否则这个部署的计划会一直执行到执行这个计划的AI死亡）。
此指挥方式对全部友军有效


### 小队指挥

在指挥自己的队伍之前，你必须有一个属于自己的小队吧？（⊙ｏ⊙）
那么如何招募队员呢 (⊙_⊙)？
只要对着AI点击G键就可以了。
是不是很简单？
在讲解如何指挥前，我们先讲讲HUD处的队友状态和数量
回到游戏中的HUD处（右下角）
SQUAD(2)的意思是小队中有2个队友
冒号后面的是队友的状态
FOLLOWING是跟随中，此时队友会一直跟着你
MOVING是移动中，代表队友正在去往指定地点的路上
HOLDING是待命中，即队友在指定地点待命中或防御指定据点
还有NO SQUAD的意思是你的小队内没有队员

下面我们就来讲讲指挥的两种方式：
通过G键
首先，长按G键，屏幕中央出现一个框
框中GO TO让你的队友去准星指定的位置
长按G的同时你也会发现屏幕中出现了一些点，点中带旗帜的是据点，没有的是载具
你可以将准星对准据点下达命令让队友去占点（前提是非己方据点）
准星对准据点后松手，队友会前往此点并占领
要让队友防守某点，操作与上面一样，长按G并将准星对准要防守的据点
在队友待命时，长按G并将视角移至脚底可以召集队友，单击G键也一样
还有针对一个敌方单位的操作
长按鼠标右键同时长按G键，可以指挥队员针对性攻击一个敌方单位
对于载具
要让队友上载具，就这样上面一样，按长G将准星对准载具：
要让队友下载具，同上：
那Commander **载具名**是什么意思？
命令的意思是招募载具内友军和即将要上载具的友军加入你的小队
通过控件指挥
点击B键，会打开一个面板，内有一个地图
左下角的REGROUP也是召集队友
直接在地图任何一处地方点一下左键，这时地图上会出现一个箭头，你的队友就会前往此地
你也可以在地图上右键任何一个据点，会出现几个选项
Move to 单纯让队友过去
attack point 过去并占领此点
如果是己方的据点，attack point会变成defend point（防守此点）

因为战役模式目前来说尚不成熟，所以本篇仅对战役模式的玩法进行简单介绍
讲到战役模式
玩家需要在每回合游戏中尽可能的获取游戏的胜利
并不断升级自己的科技树以对付不断强大的对手
每个据点被占领后每回合都可以产出金钱或科技点数（升级科技树可以用到）

下面就是操作介绍
这是存档加载界面
NEW的意思是新建存档，CONTINUE的意思是继续上一次的存档
进入游戏后，会进入这个界面

中间的是战局指挥台，右边的是点科技树的地方
在游戏的某些界面时，单击右键可以返回到上一级界面
战局指挥台的操作
在某一据点单击，点击Recruit可以生成兵力（只有某些点可以，生成兵力需要金钱）
点击某一据点，据点的蓝色小人向上弹起（多按1次会多弹起一个），这时选中一个据点，点击CAPTURE或MOVE（MOVE的出现前提是该据点是本方据点），可以移动这些兵力（每回合一个兵力只能移动一次），未被占领的据点会被占领，如果要攻击一个据点的敌人，操作一样，但是选中兵力后是点击ATTACK
当你被敌人攻击时或攻击敌人时会出现这个界面
AUTO RESOLVE可以让AI直接自动战斗
START的功能类似实时对局模式，点击后玩家会进入战斗
CANCEL的功能是你攻击敌人时的撤销动作，防止不必要的手误（被敌人攻击时该选项不可用）
升级技能树
在全局视角时，鼠标点击右边的工作台以升级科技树
有两个科技树，分别是武器和载具
点击一个项目以进行操作
鼠标拖动可以移动视角
点击一个选项会弹出一个框是关于武器的介绍
下边的UNLOCK点击可以解锁该选项，FOR后面跟着的是解锁该选项的条件
