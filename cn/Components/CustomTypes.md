# 自定义类型

## ActorFilter

## MeshSkin
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| mesh | Mesh | 此MeshSkin的Mesh（模型） |  
| materials | Material[] | 此MeshSkin模型所用材质 |  
| teamMaterial  | int | 该MeshSkin材质对应的队伍（`0`为Eagle蓝队，`1`为Raven红队） |  

# MaterialTarget
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| targetRenderer | Renderer | 目标renderer |  
| materialSlot | int | 材质数量 |
