# AutoRepairVehicleWeapon

## 描述
从[MountedWeapon](./MountedWeapon.md)类派生

一个下挂在载具的武器，用于修复载具

## 用法

参考[MountedWeapon](./MountedWeapon.md)

修复=开火，因此可以调整“开火”参数

同样需要配置`muzzles`栏位
## 注释

## 变量
| 名称 | 类型 | 描述 |
| ----------- | ----------- | ----------- |
| repairAmount | float | 每次修复的血量 |
